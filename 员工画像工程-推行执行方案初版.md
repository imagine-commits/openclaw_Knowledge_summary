# 员工画像工程 — 推行执行方案（初版）

> 版本：v1.0 | 日期：2026-05-05
> 基于：27个核心问题清单
> 目标：形成可落地的执行计划，后续根据实际生产情况迭代为完整方案

---

## 一、项目背景与目标

### 1.1 项目背景

- **单位**：中国移动数智事业部（九天公司 | 九天研究院）
- **规模**：147人
- **特点**：IT+DT+AI融合，矩阵式管理，处于数智化转型期
- **当前痛点**：人才数据分散、无统一画像、无量化筛选工具

### 1.2 核心目标

```
3个月内上线员工画像与筛选系统，支撑：
  1. 年度人才盘点
  2. 项目组队/人员调配
  3. 培训需求分析与培训规划
  4. 管理者决策支持（团队能力地图）
```

### 1.3 成功标准

| 指标 | 目标值 |
|------|--------|
| 画像覆盖人数 | ≥90%（132人+有完整基础数据） |
| 筛选准确率 | 管理者满意度≥80% |
| 系统响应时间 | 查询≤3秒 |
| 使用频率 | 每周≥10次筛选查询 |

## 二、项目组织

### 2.1 角色与分工

| 角色 | 负责方 | 主要职责 |

```
双周报：向管理层汇报进展
里程碑会：每个Phase结束评审
```

## 附录A：数据库表结构（初稿）

```sql
-- 员工基础信息表
CREATE TABLE employee (
  id            VARCHAR(20) PRIMARY KEY,   -- 工号
  name          VARCHAR(50) NOT NULL,       -- 姓名
  group_name    VARCHAR(100),              -- 组别
  position_level INT,                       -- 岗级
  position_title VARCHAR(100),             -- 岗位名称
  work_location VARCHAR(50),               -- 工作归属地
  age           INT,                       -- 年龄
  tenure_months INT,                      -- 司龄（月）
  education_level INT,                    -- 学历层级
  political_status VARCHAR(20),           -- 政治面貌
  is_youth      TINYINT(1),                -- 是否青年
  created_at    DATETIME DEFAULT NOW(),
  updated_at    DATETIME DEFAULT NOW() ON UPDATE NOW()
);

-- 人才标签表
CREATE TABLE talent_tags (
  employee_id   VARCHAR(20),
  tag_type      VARCHAR(50),    -- high_talent / it_expert / tech_stack
  tag_value     VARCHAR(100),
  source        VARCHAR(100),   -- 数据来源
  PRIMARY KEY (employee_id, tag_type, tag_value)
);

-- 绩效记录表
CREATE TABLE performance_record (
  employee_id   VARCHAR(20),
  year          INT,
  score         DECIMAL(5,2),   -- 绩效分
  grade         VARCHAR(10),     -- 绩效等级
  PRIMARY KEY (employee_id, year)
);

-- 培训记录表
CREATE TABLE training_record (
  id            INT AUTO_INCREMENT PRIMARY KEY,
  employee_id   VARCHAR(20),
  training_date DATE,
  training_hours INT,
  training_type VARCHAR(100),
  training_name VARCHAR(200)
);

-- 画像维度得分表
CREATE TABLE employee_profile (
  employee_id   VARCHAR(20) PRIMARY KEY,
  -- 基础属性维度（部分直接从employee表取，此处冗余存储方便查询）
  -- 能力维度
  tech_skill    DECIMAL(3,1),    -- 技术能力
  mgmt_skill    DECIMAL(3,1),    -- 管理能力
  communication DECIMAL(3,1),   -- 沟通能力
  innovation    DECIMAL(3,1),    -- 创新能力
  learning_ability DECIMAL(3,1), -- 学习能力
  problem_solving DECIMAL(3,1), -- 问题解决能力
  -- 绩效维度
  performance_score DECIMAL(5,2),
  project_contribution DECIMAL(3,1),
  business_value DECIMAL(3,1),
  -- 发展维度
  potential_score DECIMAL(3,1),
  development_willingness DECIMAL(3,1),
  training_hours INT,
  -- 协作维度
  collab_rating  DECIMAL(3,1),
  peer_review_score DECIMAL(3,1),
  -- 元数据
  profile_completeness DECIMAL(3,1),  -- 画像完整度
  last_updated  DATETIME,
  PRIMARY KEY (employee_id)
);
```
